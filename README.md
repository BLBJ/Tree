# Tree
彩色动画树

#### 效果示例
https://blbj.github.io/tree.github.io/

#### 使用方法
```Javascript
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>彩色大树生长动画</title>
    <script type="text/javascript" src="js/jquery-1.4.2.min.js"></script>
    <script type="text/javascript" src="js/drawTree.js"></script>
</head>
<body>

<div id="draw"></div>
<script type="text/javascript">
    //调用插件
    $('#draw').drawTree();
</script>

</body>
</html>
```
